fun <T> Flow<T>.sample(periodMillis: Long): Flow<T>(source)

Returns a flow that emits only the latest value emitted by the original flow during the given sampling period.

Example:

flow {
    repeat(10) {
        emit(it)
        delay(110)
    }
}.sample(200)
